# Frontend Engineering Challenge: Interactive Calendar Component

## 🎯 Objective
Your task is to build a polished, interactive React JS component inspired by the provided reference image. We want to see your ability to translate a static design concept into a highly functional, responsive, and user-friendly web component.

---

## 🖼 The Inspiration
Please review the attached reference image of the wall calendar. Notice:

- Clean layout  
- Visual hierarchy between imagery and dates  
- Integrated notes section  

Your component should emulate a **physical wall calendar aesthetic**, featuring:

- A prominent hero image (top or side)
- A structured calendar grid below or beside it

Use this as your creative starting point.

---

## 🛠 Core Requirements

### 1. Wall Calendar Aesthetic
- UI must resemble a physical wall calendar
- Include a **hero image section**
- Maintain balance between visual design and usability

---

### 2. Day Range Selector
- Users can select:
  - Start date
  - End date
- Include clear visual states for:
  - Start date
  - End date
  - Dates in between

---

### 3. Integrated Notes Section
- Functional notes area for:
  - Monthly memos **or**
  - Notes tied to selected date(s)
- Flexible implementation is encouraged

---

### 4. Fully Responsive Design

#### Desktop
- Preserve structured layout (e.g., side-by-side panels)

#### Mobile
- Stack layout vertically
- Ensure:
  - Smooth touch interactions
  - Usable date selection
  - Accessible notes section

---

## ✨ Creative Liberty (Stand Out!)

Go beyond the basics! Add features such as:

- Calendar flipping animations
- Theme switching based on hero image
- Holiday/event markers
- Micro-interactions & transitions

---
